PocketMCP CLI

Operate PocketBase MCP with speed and clarity.

PocketMCP gives you a focused interface to run an admin MCP server over stdio and install client configuration in one practical flow. Use it when you need predictable setup, quick onboarding, and less config friction.

Single binary workflow `serve` + `install` only Cross-platform release assets

Install in one line

Pick your shell and copy instantly.

curl -fsSL https://raw.githubusercontent.com/wil/mcp-pb/main/install.sh | bash
powershell -c "irm https://raw.githubusercontent.com/wil/mcp-pb/main/install.ps1 | iex"

Quick usage

Core commands for local admin access and client config bootstrap.

Run MCP server with serve

pocketmcp serve \
  --url http://127.0.0.1:8090 \
  --email admin@example.com \
  --password 'your_password' \
  --timeout-ms 15000

Configure clients with install 

pocketmcp install \
  --client all \
  --url http://127.0.0.1:8090 \
  --email admin@example.com \
  --password 'your_password'

Platform support
